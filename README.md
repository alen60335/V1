# 環陣行者（Ring Walker）— 陣法類銀河城 V1

依《陣法系統設計文件.md》實作的 HTML5 橫向捲軸類銀河戰士惡魔城原型。
純前端、零依賴，直接用瀏覽器開啟 `index.html` 即可遊玩（建議 Chrome / Edge）。

## 執行方式

- 直接雙擊 `index.html`，或
- 起本機伺服器：`python -m http.server 8873 --directory .` 後開 `http://localhost:8873/`

## 操作

控制採「空洞騎士式」：方向鍵移動、Space 跳、X 攻擊；元素鍵在左手 home row（A S D F）。

| 動作 | 按鍵 |
|---|---|
| 移動 | ← → |
| 跳躍 | Space 或 ↑ |
| 法杖揮擊（弱近戰） | X |
| 輸入施放元素 | A火 S水 D風 F土 |
| 起陣（施放） | Enter |
| 輸入破陣元素 | Shift + A/S/D/F |
| 發動破陣 | Shift + Enter |
| 撤銷上一個輸入 | Backspace（Shift+Backspace 撤銷破陣輸入） |
| 陣式表 | Tab |
| 暫停／靜音 | P／M |

> 元素鍵映射集中在 `js/game.js` 的 `KEY_ELEM`，攻擊鍵在 keydown 的 `KeyX` 判斷、移動在 `updatePlayer` 的 `ArrowLeft/ArrowRight`，要改綁定改這幾處即可。

## 流程（約 15～25 分鐘）

起始洞窟（教學告示）→ 走廊 → 結界一（土火，教學破陣）→ 裂谷（風系位移過谷）→
祭壇（重生點＋陣槽玉）→ 咒師走廊（對敵破陣、高台陣槽玉）→ 深層結界（五元素）→
頭目「深陣咒主」（會自癒與反噬大陣）→ 傳送門通關。

## 規則推導的重要發現（回饋給設計）

克制＋閉合規則會自然導出：**合法的純二元素陣只有 4 種**（火土、土火、水風、風水）。
另外 8 種組合中「主元素克制修飾元素」者（火風、水火、土水、風土）無法以兩元素成圓，
必須加入第三個元素才合法——這不是實作限制，而是規則的必然結果，反而讓組合有了
自然的成本梯度（強力組合需要更多陣槽）。V1 採用的建議輸入：

（下列以按鍵表示，A火 S水 D風 F土）

- 2 槽：火種 `A F`、爆破 `F A`、流療 `S D`、流體位移 `D S`
- 3 槽：延燒 `A F S`（土為間接）、追蹤 `A D F`、爆療 `S A D`、泉湧 `S A F`（火為間接）、
  回復陷阱 `F S A`、感應陷阱 `F S D`（水為間接）、衝刺 `D F A`（土為間接）、錨點 `D F S`
- 5 槽：反噬陣例 `A D S D F`（兩個風皆為間接 → 反噬）

## V1 的取捨（文件未定或本作簡化）

- 敵人不會反過來破玩家的陣（對稱破陣留待後續版本）。
- 玩家自組反噬陣目前只有視覺標記（因敵人不破陣，副效果不會觸發）。
- 破陣成功附帶：中斷施法、目標暈眩 2.5 秒、小額反傷 10（遊戲性選擇，非文件規定）。
- 「空間」元素、8 格陣槽上限成長曲線尚未實作（V1 上限 5）。

## 美術素材說明

本次 session 未連接 higgsfield MCP，全部視覺（角色、敵人、背景視差、陣法圈、UI）
以 Canvas 程式化繪製。之後若要換成生成素材，替換 `game.js` 中的
`drawPlayer`／`drawEnemy`／`drawBackground` 為圖片繪製即可（介面已集中）。

## 除錯掛鉤

隱藏分頁時 rAF 不會觸發；自動化測試可用 `window.RingWalker.step(ms)` 手動推進影格、
`RingWalker.debug` 讀取狀態、`RingWalker.resume()` 恢復正常迴圈。
