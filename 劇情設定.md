# 劇情設定 — 八苦與因果之網

> 核心：一個普通人看盡了人生八苦，最後回首才明白，這人世間宛如泥犁。
> 結構：每個角色的苦，都是**另一個角色種下的因**——沒有人是孤立的惡，也沒有人是無辜的旁觀者。

## 一、主角

**阿原**——山村的採藥人。母親年邁眼盲，妻子阿鸞，幼子小滿。
玩家操縱的「行者」，是阿原**臨終的一念清明**：他凍斃於雪峰之際，神識墜入泥犁，
走過的洞窟就是他回顧的一生。玩家自始至終不知道自己是誰——直到終殿前的最後一篇殘篇。

## 二、因果之網

```
遊商賣出《不死陣殘卷》
        │
        ▼
守陣人癡迷長生（求不得）──► 抽乾全村祭砂煉陣 ──► 山神閉眼、瘟疫入村
        │                                            │
        │             ┌──────────────┬───────────────┼──────────────┐
        │             ▼              ▼               ▼              ▼
        │      幼子小滿病倒（生）  老母衰盲（老）  醫者藏藥予親女（病）─► 獵戶喪子結怨（怨憎會）
        │             └──────┬──────┘
        │                    ▼
        │      阿原遠行雪峰採藥（為母求雪蓮、為子求藥）
        │                    │
        │                    ▼
        │      阿鸞夜夜提燈守望，化為疾風梟（愛別離）
        │                    │
        │                    ▼
        │      阿原凍斃峰頂，離雪蓮三寸（死）
        │                    │
        │                    ▼
        └──────► 執念堆疊成祖陣墟主 ◄── 泥犁成形
                             │
                             ▼
              一念清明入泥犁＝玩家（五蘊熾盛）
              回首：「八苦即八陣，人間宛如泥犁。」
```

**因果鏈驗證**（A 做了什麼 → 間接導致 B 做什麼）：
- 遊商賣書 → 守陣人變質
- 守陣人抽砂 → 瘟疫（→ 生、老、病三苦全數引爆）
- 瘟疫 → 醫者藏藥 → 獵戶結怨（怨憎會的雙方都是瘟疫的果）
- 瘟疫＋母老＋子病 → 阿原遠行 → 阿鸞化梟（愛別離是所有前因的總和）
- 阿原遠行 → 死於峰頂 → 執念成墟 → 泥犁 → 行者（玩家）
- 環形閉合：玩家破的每一座陣，都是自己一生種下的因——與陣法系統的「圓」互為隱喻

## 四、敘事載體

1. **八篇殘篇魂燈**：青白魂火，靠近即讀（讀過轉暗）；結局統計 x/8，全收有專屬結語。
2. **頭目台詞**：登場卡引言＋臨終之言（阿鸞臨終認出阿原是最重的一刀）。
3. **遊商對話**：隨擊殺頭目數推進四階段——逐步揭露他就是賣出殘卷的「因之起點」，
   他滯留泥犁行商是在償還。玩家的靈砂交易，本身就是輪迴的暗喻。
4. **開場引子**：「雪峰之上，採藥人闔上了眼。／一念清明，墜入泥犁。」
5. **結局「回首」**：全文揭示八苦即八陣；「雪，停了。」

## 五、機制與主題的互文（設計上的巧合與刻意）

- 陣法「圓形閉合」＝因果環環相扣，首尾必接。
- 「間接元素」＝無辜的橋接者——不生效果，卻承受反噬（獵戶與醫者皆是）。
- 「反噬」＝怨憎會：傷人的陣，破它的人也受傷。
- 「破陣」＝放下：不是消滅對方，而是讀懂對方的序列（一生），以克制回應。
- 最終頭目不是魔王，是自己的執念——擊敗＝放下，故結局無凱旋，只有雪停。
