> 本文件是「實際程式行為」的完整梳理，所有數值皆核對自 `js/game.js`／`js/formation.js`／`js/level.js`。

按鍵約定：**A=火　S=水　D=土　F=風**；跳=X、攻擊=C、確認=Space、破陣模式=Shift。

## 一、陣法系統

### 1. 克制環

```
水克火 → 火克風 → 風克土 → 土克水 →（回到水）
```

- **輸入時**即時檢查：同元素不能相鄰；「克制上一個元素」的元素不能接續。
- **起陣時**（Space）檢查圓形閉合：最後一個元素必須能合法接回第一個。
- 兩關都過才進入施展；閉合失敗或靈力不足時，**已輸入的序列會清空**。

### 2. 間接元素（橋接）

掃描序列三元組 (a, b, c)：若 a≠c 且 c 因克制不能直接接在 a 之後，則 b 為**間接元素**——透明化、不參與後續判定、不貢獻效果與威力。

### 3. 反噬陣

間接元素 **≥2 個且全部同元素** → 反噬陣。被破時**破陣者**受 16 點穿透傷（無視無敵時間）。
敵方反噬陣在**陣圈全部元素亮起後**，陣心會呈暗紅色供辨識。

### 4. 效果判定

- 主效果＝第 1 個有效元素；修飾＝第 2 個有效元素。
- **威力係數**＝1 +（有效元素數−2）× 0.4。注意：間接元素不算有效，所以「含橋接的 3 序列」係數仍是 ×1.0，「三個全有效」才是 ×1.4——這正是設計文件「間接元素不貢獻任何效果」的直接結果。

### 6. 施展流程與資源

```
佈陣（逐鍵輸入，全程可見）→ Space 起陣 → 施展窗口（可被打斷）→ 效果發動
```

- 施展窗口＝ 0.4 + 0.35 × 序列長度（秒）：2 元素 1.1s、3 元素 1.45s、5 元素 2.15s。
- 靈力：上限 100（靈力玉 +30），回復 7/秒；起陣消耗 7 × 序列長度，**被打斷不退還**。
- 窗口期間移動速度降為 55%，可跳躍；**受到任何傷害即中斷施展**。
- 空中也能起陣（裂谷的跳躍＋衝刺就靠這個）。

### 7. 介面輔助

- **起始標記**：所有陣圈的第一個元素放大＋金環，陣心金色指針恆指起始、旁有讀序方向箭頭——旋轉中也能判讀序列（回應「圓無正面」的判讀問題：接受任意旋轉輸入，但標記讓你不用等轉回來）。
- **輸入預覽**：輸入中即時顯示成品名稱、反噬標記與可否起陣（如「→ 追蹤火球（未成圓——尾端須能接回開頭）」）。
- **洞窟輿圖**（M）：空洞騎士式房間地圖——以「房間」為單位，走訪過才顯現輪廓（秘密房亦然）；所在房間高亮脈動、區域名稱、方向鍵平移；標記祭壇、結界、封印、頭目、未收集寶物、終點與自身。
- **環境音樂**：程序化 BGM 隨區域切換調性（洞窟／幽泉／祖陣），頭目交戰自動轉戰鬥節奏；N 靜音。

### 8. 破陣

```
Shift 進入破陣模式 → 輸入克制序列（A/S/D/F）→ Space 發動 → 不論成敗自動退出並清空
```

- 破陣序列＝把目標序列**每個元素換成克制它的元素**；陣會旋轉、沒有正面，輸入**任一旋轉週期**皆判定成功。
- 目標選擇：480px 內正在佈陣的敵人，或水平 190px 內的結界，取最近者（畫面有虛線目標圈）。
- 成功（敵陣）：中斷施法＋暈眩 2.5 秒＋直接 10 傷；若為反噬陣，自身受 16 穿透傷。
- 成功（結界）：永久破除。
- **反噬對敵陣與結界一視同仁**——破除反噬結界同樣吃 16 穿透傷（幽泉水道的通行費）。
- 失敗：只清空序列退回一般模式，無其他懲罰。
- 破陣序列輸入同樣受克制規則限制——由於克制映射保持合法性，合法陣的破陣序列必然合法。

### 2. 敵方佈陣時間線（破陣窗口）

```
佈陣期：每 0.5 秒亮一個元素（咒主 0.38 秒）——序列逐一公開
施展窗口：0.4 + 0.35 × 長度 秒——最後的反應時間
```

例：咒師的 3 元素追蹤陣，從起手到發動約 1.0s（佈陣，首元素即亮）＋ 1.45s（窗口）≈ **2.5 秒**；咒主的 5 元素反噬大陣約 1.5s ＋ 2.15s ≈ **3.7 秒**。破陣判定在「陣存在期間」（佈陣期＋窗口期）都接受——熟記陣式池的話，佈陣期就能預輸入搶破。

### 3. 頭目一・深陣咒主

- 進入深層陣殿（x>187 格）觸發現身；緩速逼近玩家（35px/秒），近身 16 接觸傷。
- 佈陣 0.38 秒/元素、施放冷卻 1.6 秒。招式輪替：

| 階段 | 招式 | 應對 |
|---|---|---|
| 全程輪替 | 三連追蹤彈（火風土，11×3） | 破陣 **S A F**，或跑位拉開 |
| 全程輪替 | 火種區（火土，12/秒） | 破陣 **S F**，或直接走出區域 |
| HP < 85% 加入輪替 | **反噬大陣**（火風水風土）：0.85 秒後在玩家位置爆裂 30 傷（半徑100，可走位躲開） | 兩難：破陣 **S A D A F** 立即中斷＋暈 2.5s＋10 傷，但吃 16 反噬；或不破、靠走位躲爆裂 |
| HP < 55%（CD 18 秒） | 自癒陣（水火風）：回復 50 | **必破**：**D S A**——省下 50 血、倒扣 10、再送暈眩輸出窗口 |

- 破陣需要 5 格破陣槽 → 兩顆陣槽玉是硬需求（深層結界也擋 5 元素，進度上天然保證）。
- 擊殺後右側封印崩解，開啟幽泉水道。

### 4. 頭目二・疾風梟主（HP 300，飛行）

懸浮於巢穴上空追蹤玩家（近戰難以搆到——陣法才是主力），佈陣 0.42 秒/元素、冷卻 1.8 秒。
**破陣使其墜地暈眩 2.5 秒**，是輸出窗口。招式輪替：

| 招式 | 內容 | 破陣 |

| 衝鋒陣（風土火） | 朝玩家直線衝撞（飛行斜衝） | **A F S** |
| 風靈彈（風水） | 三發緩速追蹤風彈，各 10 傷 | **A D**（兩鍵速破） |
| 落雷陣（土水風） | 玩家腳下佈三枚延遲地雷（1.5 秒後爆，各 16） | **F D A** |

### 5. 最終頭目・祖陣墟主（HP 420）

佈陣最快（0.34 秒/元素）、冷卻最短（1.4 秒），緩步逼近，接觸 18 傷。招式：

| 階段 | 招式 | 應對 |

## 三、成長與進度門

| 收集品 | 位置 | 效果 |
|---|---|---|
| 陣槽玉 ×3 | 祭壇室／咒師走廊高台／晶窟高台 | 施放槽與破陣槽各 +1（3 → 6） |
| 靈力玉 ×3 | 裂谷踏台／休息室／水道上層迴廊（秘） | 靈力上限 +30（→190） |
| 命玉 ×3 | 水道暗室／休息室平台／裂谷底暗道（秘） | 生命上限 +25（→175） |
| 祭壇 ×5 | 祭壇室／陣殿前／水道口／休息室／終殿前 | 全恢復＋重生點 |

進度門設計：
1. **結界一**（土火，2 元素）→ 破陣 **F S**——教學，初始 3 槽即可。
2. **裂谷** → 需風系位移（F S 或 F D A）——教學移動陣。
3. **深層結界**（土水風土火，5 元素）→ 破陣 **F D A F S**——強制收滿兩顆陣槽玉。
4. **頭目一封印** → 擊殺深陣咒主，封印崩解才能進幽泉水道。
5. **反噬結界**（火風水風土）→ 破陣 **S A D A F**，必吃 16 反傷——教學「破陣的代價」。
6. **頭目二封印** → 擊殺疾風梟主。
7. **六元素結界**（土水火風土火）→ 破陣 **F D S A F S**——強制第三顆陣槽玉。
8. **最終頭目** → 綜合驗收：讀陣、破陣、反噬抉擇、走位。
